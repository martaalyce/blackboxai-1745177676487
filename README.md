# Gerador de Link de Rastreamento

## Project Overview
O Gerador de Link de Rastreamento é uma aplicação web que permite gerar um link único de rastreamento que pode ser facilmente compartilhado. O sistema gera um link incluindo um identificador exclusivo, baseado em um timestamp e uma string aleatória, garantindo que cada link seja único.

## Installation
Para usar este projeto localmente, siga os seguintes passos:

1. Clone o repositório:
   ```bash
   git clone <URL do repositório>
   cd <nome do diretório>
   ```

2. Não há requisitos específicos de instalação além de um navegador web para executar o arquivo `index.html`.

## Usage
Para utilizar o Gerador de Link de Rastreamento:

1. Abra o arquivo `index.html` no seu navegador.
2. Um link de rastreamento será gerado automaticamente na página.
3. Você pode copiar o link clicando no botão de copiar. Uma mensagem de confirmação será exibida quando o link for copiado.

## Features
- Geração automática de um link único de rastreamento.
- Cópia do link gerado para a área de transferência com uma notificação visual.
- Design responsivo que se adapta a diferentes tamanhos de tela.

## Dependencies
Este projeto usa as seguintes bibliotecas externas:

- [Tailwind CSS](https://tailwindcss.com/) - Framework CSS para estilização.
- [Font Awesome](https://fontawesome.com/) - Biblioteca de ícones para botões.

Você não precisa instalar gerenciadores de pacotes ou bibliotecas específicas, pois todas as dependências estão incluídas através de links CDN.

## Project Structure
```plaintext
.

```

O arquivo `index.html` contém toda a lógica da aplicação, incluindo a geração de links, estilização e interação com o usuário.
